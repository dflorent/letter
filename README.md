Letter
======

Courrier HTML & CSS éditable/imprimable.

[Démo](http://dflorent.github.io/letter/)

Pré-requis
----------

- ndenv (https://github.com/riywo/ndenv)

Avant de commencer
------------------

```
echo 5.10.1 > .node-version
ndenv install
npm install
```

Lancer l'application
--------------------

```
npm start
```
